# Factory Equipment Health Dashboard

## Project Overview
This project focuses on analyzing industrial equipment telemetry data using Tableau to monitor equipment health across multiple factories and sections. The dashboard provides interactive visualizations to identify unhealthy devices, temperature-related equipment failures, and operational risk patterns.

The goal of this project is to transform raw telemetry data into actionable insights for predictive maintenance and equipment monitoring.

---

## Features
- KPI cards for device health metrics
- Factory-wise unhealthy equipment analysis
- Section-wise equipment health heatmap
- Scatter plot with linear trend analysis for temperature vs unhealthy devices
- Interactive dashboard filters
- Device-level analysis using telemetry data
- Professional dashboard design with dark theme

---

## Tools & Technologies Used
- Tableau Public
- Data Visualization
- Data Analytics
- Dashboard Design
- Telemetry Data Analysis

---

## Dashboard Insights
The dashboard helps identify:
- Factories with the highest unhealthy equipment count
- Relationship between temperature and device failure
- High-risk operational sections
- Overall equipment health status
- Device performance trends

---

## KPI Metrics Included
- Total Devices
- Unhealthy Devices Count
- Average Temperature
- Healthy Device Percentage
- Total Factories
- Total Sections

## Dataset Information
The dataset contains industrial telemetry information including:

- Device Type
- Factory
- Section

## Project Objective
To build an interactive analytics dashboard that supports equipment monitoring, operational analysis, and predictive maintenance decision-making in industrial environments.
